# gmodel


一个基于leveldb的内容存储模块，适合在超低配置的服务器上运行具有大量文章数据的网站，比如：新闻、词典等类型的网站。


## 特点

- 几乎不占用内存，启动快

- 支持增删改查等基本操作，支持按分类查找




## 基本信息：

- gmodel：包装了下面的文章和分类，对外一般用这个，如果需要防止网站被采集，还需要使用下面的 idmgr 模块

- article： 文章

- tag：  分类

- kvstore：kv存储组件

- utils：实用函数

- idmgr：将整型ID和字符串ID映射，避免通过递增ID就可以遍历网站


- remote：提供远端访问的API服务器，方便 website/admin/spider 分离
